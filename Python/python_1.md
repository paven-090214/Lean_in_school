## 1.1장
### 데이터 과학 : 데이터에서 과학적 방법으로 정보나 지식을 추출하는 방법
- 데이터 분석
- 구조 파악
- 알고리즘
- 프로글밍
- 문제해결과 지식
> 짐 그레이 박사 :
 - 과학의 네 번째 파라다임으로 데이터 중심의 과학이 등장하고있다고 보았다.
 - 컴퓨터를 이용한 계산 과학이 과학 연구의 방법론으로 자리를 잡았다고 분서하였다.
> 빅데이터를 활용한 사례
### 데이터 처리 과정
## 데이터 과학은 데이터를 확보하여 응용 분야에 사용할 수 있는 정보로 만들기 위해 다양한 과정이 존재한다.
1. 데이터 수집
2. 데이터 정제
3. 탐구와 분석
4. 학습 모델 구축
5. 모델 배포
