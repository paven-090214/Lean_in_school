# 3장
## 용어
- 수식 : 피연산자들과 연산자들의 조합
- 연산자 : 어떤 연산을 나타내는 기호
- 거듭제곱
    - 다른 연산자들보다 높은 우선순위를 갖는다.
    - 오른쪽에서 왼쪽으로 계산된다.
### 파이썬에서의 나눗셈
- / : 실수 나눗셈
    - 7 / 4 = 1.75
- // : 정수 나눗셈의 몫
    - 7 // 4 = 1
