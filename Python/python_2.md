## 변수
- 변수 : 컴퓨터의 메모리 공간에 이름을 붙이는 것이다.
  - 예약어는 변수의 이름으로 사용할 수 없다.
  - 변수의 역할을 잘 설명하는 이름으로 지어야 좋다.
- 객체 : 문자열이나 데이터 수치형 데이터를 의미한다.

```python
weight = 51.4
print(weight)
```
실행결과
```
51.4
```
## 동시 할당
```
x, y = 10, 20 # x, y에 각각 10, 20을 동시에 할당
result = x + y
print(result)
```
실행결과
```
30
```
## 문자열 
```
a = "안녕"
b = "하세요"
print (a + b)
```
실행결과
```
'안녕하세요'
```
## BMI 계산하기

weight = 53.3

print(f'{BMI:.4f}')
