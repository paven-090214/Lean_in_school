## 용어
- 변수 : 컴퓨터의 메모리 공간에 이름을 붙이는 것이다.
  - 프로그램 코드의 가동성이 좋아진다.
  - 예약어는 변수의 이름으로 사용할 수 없다.
  - 변수의 역할을 잘 설명하는 이름으로 지어야 좋다.
- 객체 : 문자열이나 데이터 수치형 데이터를 의미한다.
- 자료형
    - 정수형(int)
    - 실수형(float)
    - 문자열(tr)
    - 부울형(bool)
    - 리스트(list)
- 정적 형결정 : 어떤 변수가 담을 수 있는 자료형이 번역할 때 고정된다.
    - C
- 동적 형결정 : 프로그램이 동작하면 변수에 값을 넣을 때 정해진다.
    - 파이썬
- 메소드 : 객체 + 명령() 형식의 구조에서 .연산자로 구분된 명령을 내일 때 이 명령을 메소드라고 한다.
## 변수
```python
weight = 51.4
print(weight)
```
실행결과
```
51.4
```
## 소숫점 계산
```
print(0.1 + 0.1 == 0.2)
print(0.1 + 0.1 + 0.1 == 0.3)
```
실행결과
```
True
False
```
## 동시 할당
```
x, y = 10, 20 # x, y에 각각 10, 20을 동시에 할당
result = x + y
print(result)
```
실행결과
```
30
```
## 문자열

print (a + b)

'안녕하세요'

## 출력 방법 & 문자열 입력 -> 정수로 변환
```
a = int(input("첫 번째 정수를 입력하세요 : "))
b = int(input("두 번째 정수를 입력하세요 : "))
print(f"{a}와 {b}의 합은 {a+b}입니다.")
print("%d와 %d의 합은 %d입니다."%(a, b, a+b))
print("{}와 {}의 합은 {}입니다." .format(a, b, a+b))
```
실행결과 (30, 10 대입)
```
30와 10의 합은 40입니다.
30와 10의 합은 40입니다.
30와 10의 합은 40입니다.
```
