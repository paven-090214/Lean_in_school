# W01
## JSP
- JSP(Java Server Pages)는 동적인 웹 페이지를 개발하기 위한 웹 프로그래밍 기술이다.
- 자바(Java)언어를 사용하여 서버 측에서 웹 페이지를 생성해 웹 브라우저로 전송한다.
## 장점
- 짧은 코드로 동적인 웹 페이지를 생성할 수 있습니다.
- 기본적인 예외는 자동으로 처리해줍니다.
- 많은 확장 라이브러리를 사용할 수 있습니다.
- 스레드 기반으로 실행되어 시스템 자원을 절약해줍니다.
## JSP의 기본 구조
1. <%! %> : 선언부 -> 멤버변수나 메서드 선언
2. <%= %> : 표현식 -> 변수 출력, 메서드 호출
3. <% %> : 스크립틀릿 -> Java코드 작성
4. <%@ %> : 지시어 -> JSP 페이지를 자바(서블릿) 코드로 변환할때 필요한 정보를 JSP 엔진에 알려준다.(스크립트 언어나 인코딩 방식 등을 설정)
   - page 지시어 : JSP 페이지에 대한 정보를 설정
   - include 지시어 : 외부 파일을 현재 JSP 페이지에 포함시킴
   - taglib 지시어 : 표현 언어에서 사용할 자바 클래스나 JSTL을 선언

# 내장객체
1. reuest 객체
  - 클라이언트의 요청을 받거나 웹 브라우저에 대한 정보 혹은 요청에 대한 정보를 읽을 때 사용합니다.
2. respone 객체
  - 요청에 대한 응답을 웹 브라우저로 보낼 때 사용합니다.
  - 페이지 이동이나 응답 헤더를 추가할 때 사용합니다.
3. out 객체
  - 변수 등의 값을 웹 브라우저에 출력할 때 주로 사용합니다.
4. application 객체
  - 웹 애플리케이션을 구성하는 모든 JSP에서 접근 가능한 객체로, 웹 애플리케이션에 대한 설정값을 저장할 때 주로 사용합니다.
5. exception 객체
  - 예외 처리를 위해 사용합니다.

request = 클라이언트 → 서버 로 보내는 요청
response = 서버 → 클라이언트 로 보내는 응답
