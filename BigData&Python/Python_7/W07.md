## 주관식 문제
1. 프로그래밍 언어에서 사용되는 용어인 **`함수`** 는 우리가 반복적으로 사용하는 코드를 묶은 것으로 코드의 덩어리와 같다.
2. 프로그래밍 언어에서 사용되는 용언 **`객체`** 은 코드 중에서 독립적인 단위로 분리할 수 있는 조각이다.
3. 파이썬 언어에서 말하는 **`모듈`** 은 프로그램이 일부를 가지고 있는 독립적인 파일을 말한다.
4. 함수는 파이썬 키워드 **`def`** 를 이용하여 정의한다. 이 키워드에 이어서 함수의 이름을 적은 후에 **`소괄호()`** 와 **`:`** 를 붙인다.
5. 함수를 호출하려면 **`함수의 이름`** 과 **`소괄호()`** 를 이용한다.
6. 함수 호출시 전달되는 실제 값을 **`인자(argument)`** 라고 하고, 함수 내부에서 전달받는 변수를 **`매개변수(parameter)`** 라고 한다.
7. 함수로부터 되돌아오는 값을 **`반환 값(return value)`** 이라고 한다.
8. 함수 안에서 생성되는 변수를 **`지역변수`** 라고 한다.
9. 함수 내부에서 전역변수를 수정하려면 **`global`** 키워드를 사용해야한다.
10. 파이썬에서는 함수의 매개변수가 기본값을 가질 수 있다. 이것을 **`디폴트 인자(default argument)`** 이라고 한다.
11. **`재귀함수`** 란 함수 내부에서 자기 자신을 호출하는 함수를 말한다. 

 ## 심화문제
 ### 가장 큰 값, 작은 값 구하기
```python
n1 = int(input('숫자 : '))
n2 = int(input('숫자 : '))
n3 = int(input('숫자 : '))

def max_min(a,b,c) :
    num_1, minNum = (a, b) if a >b else (b, a)
    print(f'가장 큰 숫자 : {num_1 if num_1 > c else c}, 가장 작은 숫자 : {minNum if minNum < c else c}')

max_min(n1, n2, n3)
```

for _ in range(0, 60, 10) :

### 팩토리얼 함수
```python
def factorial(userNum) :
    x = 1
    for _ in range(userNum, 0, -1) :
        x *= _
    return x
print(factorial(5))
```
### 두 점 사이 거리 구하기
```python
import math
def distance(x1, y1, x2, y2) :
    print(f'({x1, y2})과 ({x2, y2}) 사이의 거리는 {math.sqrt((x2 - x1) ** 2 + (y2 - y1) ** 2)}')

x1 = int(input('x1의 좌표 : '))
y1 = int(input('y1의 좌표 : '))
x2 = int(input('x2의 좌표 : '))
y2 = int(input('y2의 좌표 : '))
distance(x1, y1, x2, y2)
```
### all 실행
```python
def my_all(iterable):
    for x in iterable:
        if not x:
            return False
    return True
```
> 거짓(False)인 값이 하나라도 나오면 바로 `False`  
> 끝까지 봤는데 거짓이 한 번도 없으면 `True`
### any 실행
```
def my_any(iterable):
    for x in iterable:
        if x:
            return True
    return False
```
> 참(True)인 값이 하나라도 나오면 바로 `True`  
> 끝까지 봤는데 참이 한 번도 없으면 `False`
