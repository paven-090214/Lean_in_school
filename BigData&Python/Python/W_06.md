## 주관식
1. 횟수 제어 반복 -> **`for`**
2. 조건 제어 반복 -> **`while`**
3. 프로그래밍에서 반복은 흔히 올가미 같은 동그란 고리를 의미하는 **`루프(loop)`** 라고 한다.
4. 파이썬에서 들여쓰기를 하여야 한다.  -> **`블록`**
5. 파이썬에서 들여쓰기 시작 특수문자 -> **`:`**
6. **`range()`** 함수는 for반복문에서 자주 사용되며, 연속적인 숫자의 시퀀스를 생성한다.
7. 조건 제어 루프에서 무한히 반복하며 종료되지 않는 일 : **`무한 루프`**
8. while 문의 조건이 참이면 조건문 다음의 블록이 반복적으로 실행된다.
9. **`break`**은 루프를 강제로 빠져나올 때 사용한다.
10. **`continue`** 는 루프를 빠져나가지 않고 아래쪽 문장을 건너뛰기만 한다.
11. **`format`** 메소드는 플레이스 홀더를 이용하여 원하는 문자나 숫자를 적절한 포맷으로 출력할 수 있다.
12. 소숫점 지정 num:.2f의 f는 **`부동소수`** 를 의미한다.
## 심화문제
### 1부터100까지 홀수 합(while, for)
```python
i = 0
s = 0
while i <= 100 :
    if i % 2 == 1 :
        s += i
    i += 1
print(s)

a=0
for _ in range(101) :
    if _ % 2 == 1 :
        a += _
print(a)
```
### 시작 정수부터 끝 정수 더하기
```python
s = 0
a = int(input('첫 번째 정수 : '))
b = int(input('두 번째 정수 : '))
for _ in range(a, b +1) :
    s += _
print('합 :', s)
```
### while True, if break
```python
print('메뉴 --------------------\n1. 치킨\t2. 피자\t3. 햄버거')
while True :
    a = int(input('메뉴 번호 : '))
    if a == 1 :
        print('치킨')
        break
    elif a == 2 :
        print('피자')
        break
    elif a == 3 :
        print('햄버거')
        break
    else :
        print('번호 다시 입력')
```
### 별 삼각형 출력
```python
a = int(input('숫자 입력 : '))
for i in range(1, a + 1) :
    for s in range(a - i) :
        print(' ',end='')
    for j in range(i) :
        print('*', end='')
    print('\n')
```
