## 매핑 규칙 : 개념적 설계 -> 논리적 설계 (중요)
> 총 7단계 존재  
1단계 : 정규 엔티티 -> 테이블  
2단계 : 약 엔티티 -> 실선으로 연결  
3단계 : 1 : 1 관계 -> NULL값 최소화  
4단계 : 1 : N 관계 -> N쪽으로 복사  
5단계 : M : N 관계 -> 별도의 테이블로 분리, 실선 연결  
6단계 : 다중값  
          - >  별도의 테이블로 분리  
          -  > 실선으로 연결  
7단계 : 3진관계(마름모에 연결된 사각형이 3개)  
 - 별도의 테이블로 분리
 - 각각각 기본키로 연결
